This is the intern project for Nufit Media, a lead generation startup at Cogo Lab.
neighbor.py: list all zips that are within 5mi from the target zip for every zip code in the USA
webScraper.py: scrap the website with "beautiful soup" and find zip codes for all walmarts across USA
zipToNeighbor.py and cityToNeighbor.py: takes in a file of zip codes/city names, then spit out all the neighbor zip codes (including itself)
